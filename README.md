# Breathe ESG Backend

This is a backend project built using Django and Django REST Framework for managing ESG (Environmental, Social, and Governance) data.

The project stores company details, emission records, and calculates ESG scores through APIs.

## Features

- Company API
- Emission Records API
- ESG Score Calculation
- MySQL Database Integration
- REST APIs using Django REST Framework
- CSV data support

## Technologies Used

- Python
- Django
- Django REST Framework
- MySQL
- Pandas
